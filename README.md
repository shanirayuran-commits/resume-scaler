# Resume Scaler

Resume Scaler is a web application that helps users analyze and optimize their resumes for better ATS (Applicant Tracking System) performance.

It provides feedback on formatting, keyword usage, and overall resume structure to improve job application success rates.

---

## 🚀 Features

- Resume upload and analysis
- ATS compatibility scoring
- Keyword optimization suggestions
- Clean UI for easy interaction
- Instant feedback system
